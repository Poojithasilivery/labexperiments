# 📚 Online Book Store / Shopping Cart Web Application

This repository contains a Web Technology Lab experiment developed as part of academic coursework. The project demonstrates the implementation of a simple and user-friendly online bookstore web application where users can browse available books, login securely, and manage their shopping cart.

The main objective of this project is to understand the concepts of front-end web development, page navigation, and structured document validation using XML/DTD.

## 🚀 Features
- User-friendly Home Page
- Secure Login Page
- Book Catalog / Store Page
- Shopping Cart Page
- Structured XML validation using DTD
- Easy navigation between pages
- Responsive and attractive design

## 📌 Technologies Used
- **HTML5** – For creating web page structure  
- **CSS3** – For styling and layout design  
- **JavaScript** – For interactivity and validation  
- **XML** – For storing structured data  
- **DTD** – For validating XML structure  

## 📂 Project Files
- **index.html** → Home page of the application  
- **login.html** → Login page for user authentication  
- **bookstore.html** → Displays available books/catalog  
- **cart.html** → Shopping cart details  
- **bookstore.dtd** → DTD file for XML validation  

## 🎯 Objective
The objective of this experiment is to learn:
- Designing multiple interconnected web pages  
- Creating attractive UI using HTML and CSS  
- Implementing basic validation using JavaScript  
- Understanding XML and DTD concepts  
- Developing a mini real-time web application
